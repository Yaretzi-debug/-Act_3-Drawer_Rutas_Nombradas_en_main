# -Act_3-Drawer_Rutas_Nombradas_en_main
 Act 3 Drawer Rutas Nombradas en main

Drawer ( con 4 opciones)
encabezado: imagen avatar de reposteria, (https://raw.githubusercontent.com/Yaretzi-debug/imagenes-para-flutter-6to-I-fecha-11-feb-2026/refs/heads/main/chef.webp)     , nombre empresa, dirección, teléfono, correo,
Listile: icono, texto y acción para abrir  pagina según opción de listile, (productos, postres, chefs)
En cada pagina una imagen centrada de 200x200 desde la red  (desde github)
Utilizar rutas nombradas para navegar entre paginas desde el drawer (rutas nombradas)
esteticas:grises, rosas pasteles combinado con emojis y letras cursivas con bordes brillantes en las imagenes.
con botones funcionales de su barra de navegacion, con un boton de decoracion para cambiar el estilo de la pagina
trabajar con firebase studio
En readme escribir el prompt para generar la app en un solo archivo según lo anterior + las imágenes correspondientes,
probar en dartpad antes de trabajar en firbase studio.
en firebase studio el main.dart  rutas nombradas + carpeta  "LasPaginas" y dentro de ellas los archivos correspondientes de dart.
