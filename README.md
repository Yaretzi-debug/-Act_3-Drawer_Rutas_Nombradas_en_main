# Bunny Parfait Bakery 🎀

Una aplicación de Flutter con temática 'Light Coquette & Bunny Parfait' que demuestra una arquitectura limpia, navegación con rutas nombradas y una interfaz de usuario atractiva.

## ✨ Características

* **Arquitectura limpia:** El proyecto está organizado en carpetas para separar las páginas de la configuración principal de la aplicación.
* **Rutas nombradas:** La navegación se gestiona mediante rutas nombradas, lo que facilita el mantenimiento y la legibilidad del código.
* **Drawer dinámico:** El menú lateral (Drawer) es un widget reutilizable que permite una navegación fluida entre las diferentes secciones de la aplicación.
* **Diseño 'Light Coquette':** La interfaz de usuario combina un tema claro con elementos femeninos y dulces, como colores pastel, tipografía elegante y detalles de conejitos.
* **Componentes interactivos:** La aplicación utiliza botones funcionales y tarjetas de presentación para mostrar información de forma atractiva.
